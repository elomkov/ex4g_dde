DDE сервер для взаимодействия Quik --> Внешние подсистемы

Возможности выгрузки:
1. Взаимодействие с RDBMS Oracle:
    - выгрузка таблицы всех сделок
    - выгрузка таблицы текущих значений параметров
2. Выгрузка в TXT: 
    - выгрузка таблицы всех сделок
3. Поддержка сторонних плагинов:
   - выгрузка таблицы всех сделок

Возможности сервера:
1. Режим многопоточности:
    - вкл./выкл.
    - количество активных потоков
   
   В режиме многопоточности при количестве активных потоков > 1 вывод в сторонние системы
   асинхронный.
   Для получения синхронного вывода необходимо либо выкл. режим многопоточности, либо установить
   число активных потоков == 1.

2. Настраиваемое количество обрабатываемых записей в контексте, иными словами сколько в оперативной памяти
   за раз будет обрабатываться поступивших записей перед отправкой во внешнюю систему.

3. Поддержка самописных плагинов, с возможностью указания имени библиотеки и вызываемых функций

4. Настройка объектов импорта и их параметров при взаимодействии с RDBMS Oracle

Содержимое параметров и их значений настраивается в ex4g_dde.ini

Пример настроек вывода из Quik в TXT файл:
1. DDE сервер: ex4g_dde
2. Рабочая книга: DDE_IMP
3. Лист: IMP_TRN_TXT
4. Все галочки снять
5. Нажать кнопку "Начать вывод"
